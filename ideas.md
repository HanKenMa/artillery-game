# Brainstorm Desain - Artillery Physics Game

## Konsep Game
Game fisika berbasis web yang menampilkan simulasi peluncuran proyektil dengan mekanika parabola. Pemain mengontrol sudut, kekuatan, dan gravitasi untuk mengenai target yang bergerak acak di arena.

---

## Ide Desain 1: Futuristic Sci-Fi Aesthetic
**Design Movement:** Cyberpunk + Sci-Fi Gaming UI  
**Probability:** 0.08

### Core Principles
- Neon glow effects dengan warna biru, ungu, dan cyan
- Geometric shapes dan grid patterns sebagai background
- Minimalis namun high-tech appearance
- Real-time data visualization dengan HUD-style display

### Color Philosophy
- Primary: Neon cyan (#00FFFF) dan electric purple (#9D00FF)
- Secondary: Deep space black (#0A0E27) dengan aksen silver (#E0E0E0)
- Emotional intent: Futuristik, energik, dan immersive

### Layout Paradigm
- Asymmetric split-screen: game canvas di kiri, control panel di kanan
- Floating HUD elements dengan glassmorphism effect
- Vertical stacked stats panel dengan animated counters

### Signature Elements
- Animated grid background dengan parallax effect
- Glowing particle effects saat peluncuran
- Neon border glow pada target dan projectile
- Holographic-style typography dengan letter-spacing

### Interaction Philosophy
- Smooth slider controls dengan glow feedback
- Projectile trail dengan neon gradient
- Explosion effect dengan particle burst animation
- Real-time angle preview dengan laser pointer

### Animation
- 300ms ease-out transitions untuk UI elements
- Particle system untuk explosion dengan 0.5s duration
- Smooth projectile movement dengan 60fps
- Pulsing glow effect pada target (infinite loop, 1.5s)

### Typography System
- Display: "Orbitron" atau "Space Mono" (bold, futuristic)
- Body: "Roboto Mono" (clean, technical)
- Hierarchy: Large bold display untuk score, monospace untuk values

## Ide Desain 2: Retro Arcade Game Style
**Design Movement:** 8-bit/16-bit Arcade Nostalgia  
**Probability:** 0.07

### Core Principles
- Pixel art aesthetic dengan limited color palette
- Chunky, bold typography dengan retro fonts
- Simple geometric shapes dengan high contrast
- Playful, energetic vibe dengan bright colors

### Color Philosophy
- Primary: Bright yellow (#FFFF00) dan hot pink (#FF1493)
- Secondary: Electric blue (#0080FF) dengan cream (#FFFACD)
- Emotional intent: Nostalgic, playful, arcade-like

### Layout Paradigm
- Classic arcade cabinet layout dengan centered canvas
- Scoreboard di top dengan pixelated numbers
- Control buttons di bottom dengan retro styling
- Decorative pixel borders dan scanline effects

### Signature Elements
- Pixel art cannon dan target sprites
- Retro arcade font untuk all text
- Scanline overlay effect pada canvas
- Pixelated explosion animation

### Interaction Philosophy
- Chunky button presses dengan satisfying click feedback
- Beep/boop sound effects (visual representation)
- Pixelated trail effect
- Simple, direct controls tanpa complexity

### Animation
- Snappy 100-150ms transitions
- Pixel-perfect movement (no smooth curves)
- Blinking cursor effect pada active controls
- Retro "game over" style animations

### Typography System
- Display: "Press Start 2P" atau "VT323" (pixel perfect)
- Body: "Courier New" (monospace, retro)
- All caps untuk emphasis

## Ide Desain 3: Modern Minimalist + Physics Visualization
**Design Movement:** Scientific Visualization + Modern Minimalism  
**Probability:** 0.06

### Core Principles
- Clean, spacious layout dengan generous whitespace
- Emphasis pada data visualization dan physics accuracy
- Subtle gradients dan soft shadows
- Educational yet engaging interface

### Color Philosophy
- Primary: Soft indigo (#4F46E5) dan teal (#14B8A6)
- Secondary: Neutral grays (#F3F4F6, #6B7280) dengan white background
- Emotional intent: Trustworthy, educational, sophisticated

### Layout Paradigm
- Centered canvas dengan side panels
- Graph visualization untuk trajectory prediction
- Floating control cards dengan soft shadows
- Physics info panel dengan real-time calculations

### Signature Elements
- Trajectory prediction curve dengan animated dashes
- Smooth gradient backgrounds
- Soft-edged buttons dengan hover states
- Animated physics equations display

### Interaction Philosophy
- Smooth slider interactions dengan visual feedback
- Trajectory preview before shooting
- Real-time physics parameter display
- Gentle animations pada state changes

### Animation
- 200-300ms ease-in-out transitions
- Smooth curve drawing untuk trajectory (1s duration)
- Fade in/out effects untuk UI elements
- Subtle scale animations pada hover

### Typography System
- Display: "Poppins" atau "Outfit" (modern, clean)
- Body: "Inter" (highly readable, minimal)
- Hierarchy: Weight variation (300, 500, 700)

## Pilihan Desain: **Ide 1 - Futuristic Sci-Fi Aesthetic**

Alasan: Ide ini menawarkan visual yang paling menarik dan engaging untuk game, dengan efek neon yang eye-catching dan HUD-style display yang cocok untuk game fisika. Kombinasi cyberpunk aesthetic dengan interactive elements akan membuat game terasa modern dan immersive.

### Design Direction yang Dipilih:
- **Warna Utama:** Neon cyan (#00FFFF), electric purple (#9D00FF), deep space black (#0A0E27)
- **Typography:** "Space Mono" untuk display, "Roboto Mono" untuk body
- **Layout:** Asymmetric dengan game canvas di tengah, control panel di kanan
- **Effects:** Neon glow, particle effects, glassmorphism, animated grid background
- **Animation:** Smooth transitions, particle bursts, pulsing effects
